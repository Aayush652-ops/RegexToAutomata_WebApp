# RegexToAutomata_WebApp
An interactive visualization tool for converting regular expressions into ε-NFA using Thompson’s construction. The application demonstrates step-by-step automaton construction, supports conversion to DFA via subset construction, and allows simulation of input strings, helping users understand the relationship between regex and finite automata.

## ⚠️ Best Viewing Experience

- Recommended zoom level: **50%**
- Reason: Ensures proper visibility of the full automaton graph and UI controls without clipping.

> Tip: Use `Ctrl + -` (Windows) or `Cmd + -` (Mac) to adjust zoom.
